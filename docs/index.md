# Welcome to the Open Radar Data Documentation

The **Open Radar Data (ORD) API** is a service developed within the EU- and EUMETNET-funded [RODEO project](https://rodeo-project.eu/). It provides access to EUMETNET OPERA weather radar data and products, enabling users to access radar data for visualisation, analysis, and integration into other systems.

**MeteoGate** serves as the one-stop shop for meteorological and hydrological products and data. MeteoGate is run by [EUMETNET](https://www.eumetnet.eu/). More information: [meteogate.eu](https://meteogate.eu/).

---

## Who Should Use This?

This guide is designed for:

- **Data Consumers:** Users accessing public radar data through open APIs
- **Data Publishers:** EUMETNET Members contributing radar data via the Ingest API
